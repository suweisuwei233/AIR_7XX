合宙有四种型号的gps模块：Air530、Air530H、Air530Z、AIR510U
《530》是对应Air530的demo
《530H》是对应Air530H的demo
《530Z》是对应Air530Z的demo
《510U》是对应Air510U的demo
《自适应》是可以自适应Air530、Air530H、Air530Z、AIR510U四种gps模块的demo
《other》是使用非合宙的gps模块的demo

如果不确定自己使用的Air530、Air530H、Air530Z、AIR510U中的哪一款模块，就参考《自适应》的demo
如果确定自己使用的Air530、Air530H、Air530Z、AIR510U中的哪一款模块，就参考对应的demo
如果使用的不是合宙的gps模块，就参考other的demo
